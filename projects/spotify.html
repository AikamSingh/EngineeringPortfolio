Spotify Music Controller

A physical desk-side controller for Spotify — rotary encoder navigation, e-ink display for album art, and local API integration. No phone required.

Status In Progress
Platform ESP32
Languages C / C++
🚧

Work in Progress

This project is actively under development. This page will be updated with schematics, firmware details, and build photos as the project progresses.

The Idea

I wanted a physical controller for Spotify that lives on my desk — something with real tactile feedback, a display for album art, and no dependency on pulling out a phone or alt-tabbing to a browser. The controller talks to the Spotify Web API over Wi-Fi, so it works with any device playing on the account.

The ESP32 was the natural choice — it has Wi-Fi built in, enough GPIO for a rotary encoder and several buttons, and a well-supported Arduino/ESP-IDF ecosystem for display drivers and HTTP clients.

🎵

Design Goals

Play/pause, skip, volume control via rotary encoder. Album art on e-ink display. Track name and artist in text. Low power when idle. Small enough to live next to a keyboard without getting in the way.

Status

Currently working through Spotify Web API OAuth flow on ESP32, display driver integration, and enclosure design. Updates coming.
