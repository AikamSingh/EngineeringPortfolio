# Engineering Portfolio

Personal portfolio website, built to showcase engineering projects — embedded systems, PCB design, and software — through detailed case studies rather than a simple project list.

Live at [aikamsingh.com](https://aikamsingh.com), hosted on Netlify.

## Structure

- `index.html` / `style.css` / `main.js` — homepage
- `projects/` — individual project case study pages (solar car steering wheel, smartwatch, Spotify dashboard)
- `assets/` — images, PCB renders, schematics, and video used across project pages
- `netlify/` — Netlify functions and deployment config
- `success.html` — contact form submission confirmation page

## Project page format

Each case study covers: the problem being solved, design requirements/constraints, design process (diagrams, schematics, CAD), key technical decisions, challenges encountered, and measurable results.
